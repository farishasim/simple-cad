Menggambar Garis

1. Klik tombol Line (Default sudah garis)
2. Klik dua koordinat pada canvas

Mengubah Panjang Garis

1. Klik tombol Move untuk mengubah atau memindahkan garis 2. Klik salah satu titik garis pada canvas (Lingkaran di ujung garis)
2. Drag ke posisi/koordinat lain di canvas 3. Drop titik yang telah dipilih maka garis akan berubah panjangnya

Menggambar Persegi

1. Klik tombol Square
2. Klik satu koordinat pada canvas sebagai titik pojok kiri atas persegi

Mengubah Panjang Sisi Persegi

1. Klik tombol Move
2. Klik salah satu titik persegi pada canvas
3. Drag dan drop titik yang telah dipilih untuk mengubah panjang sisi persegi

Menggambar Polygon

1. Klik tombol Polygon
2. Klik sebanyak n titik koordinat pada kanvas, sesuai dengan banyaknya titik pada polygon yang ingin dibuat
3. Klik tombol Draw Polygon

Mengubah Bentuk Polygon

1. Klik tombol Move
2. Klik salah satu titik polygon pada canvas
3. Drag dan drop titik yang telah dipilih untuk mengubah bentuk polygon

Mengganti Warna Bangun Datar

1. Klik tombol Select
2. Pilih warna yang diinginkan pada dropdown color
3. Pilih Bangun Datar yang ingin dirubah warnanya

Menyimpan Gambar

1. Klik tombol Save
2. Kemudian akan terdownload sebuah file json (hasil Save File)

Memuat Gambar

1. Klik tombol Choose File
2. Pilih lokasi file
3. Klik tombol Load
